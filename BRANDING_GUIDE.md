# 🎨 EduFlow Brand Guidelines

## Brand Identity

**EduFlow** is a modern, professional Learning Management System designed for educational institutions.

---

## 📚 Logo

### Logo File

- **Location**: `/logo.svg` (root directory)
- **Favicon**: `/frontend/public/favicon.svg` (browser tab icon)
- **Dimensions**: 200x200px (scalable SVG)

### Logo Design

- **Primary Color**: Deep Blue (#0D47A1)
- **Primary Gradient**: Light Blue (#64B5F6 to #42A5F5)
- **Accent Color**: Orange (#FF6B35)
- **Design Elements**:
  - Book icon (represents education)
  - Flowing lines (represents smooth learning flow)
  - Center accent (represents focus/progress)

### Color Palette

```
Primary Blue:      #0D47A1 (Deep Blue)
Light Blue:        #42A5F5 (Accent Blue)
Gradient Blue:     #64B5F6 (Light Blue)
Accent Orange:     #FF6B35 (Energy/Progress)
Dark Accent:       #1565C0 (Border/Text)
```

---

## 🎯 Brand Colors

| Color             | Hex Code | Usage                            |
| ----------------- | -------- | -------------------------------- |
| **Primary Blue**  | #0D47A1  | Main background, primary buttons |
| **Light Blue**    | #42A5F5  | Secondary buttons, accents       |
| **Gradient Blue** | #64B5F6  | Hover states, highlights         |
| **Accent Orange** | #FF6B35  | CTAs, progress indicators        |
| **Dark Blue**     | #1565C0  | Text, borders                    |
| **Light Gray**    | #F5F5F5  | Backgrounds, subtle elements     |
| **White**         | #FFFFFF  | Content areas, cards             |

## 🎨 Typography

- **Headlines**: Bold, professional sans-serif
- **Body Text**: Clean, readable sans-serif (default system font)
- **Code**: Monospace (for technical documentation)

---

## 📐 Logo Spacing

When using the logo, maintain clear space around it:

- Minimum padding: 10% of logo height/width
- Do not distort or rotate the logo
- Keep aspect ratio 1:1
